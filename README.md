### Github 上传100M 以上文件解决方案

- 下载git-lfs-windows

- 在git bash here 中运行 git lfs install

- git lfs track '*.pdf'

- git add .gitattributes
 
- git add .

- git commit -m 'ebook'

- git pull --rebase origin main

- git push origin main

### Markdown语法

- #一级标题

- ##二级标题

- ###三级标题

- 字体加粗两个*

- 斜体一个*

- 斜体加粗三个*

- 删除线两个~

- 引用>

- 分割线 三个或以上-或者*

- ！[图片alt](图片地址,"图片title")

- [超链接名]（超链接地址）

- 列表-+*

- 表格 表头|表头|表头 内容|内容|内容

- 代码块 反引号``` 反引号```
